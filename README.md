# Clipboard Studio

Clipboard Studio is an open-source clipboard management application to help you keep track of your clipboard text history.

## Features

- Import from Windows Clipboard History
- Automatic Clipboard Capture
- System Tray Support
- Unlimited History
- Export to File

## Installation
